# Municipal Archive – Frontend Prototype

This project is a frontend prototype of a Municipal Archive web information system.  
It was developed using **React**, **Vite**, and **TailwindCSS** as part of an academic assignment for the “Web Systems in Information Systems” laboratory.

## Features
- User authentication (login)
- Document upload form (mock processing)
- Search with filters (year, department)
- Search results list
- Document preview with metadata
- Error, loading, and empty states

## Tech Stack
- React
- Vite
- TailwindCSS
- Mock data (no backend)

## Project Structure
- `/src/pages` – Application pages
- `/src/components` – Reusable UI components
- `/src/mockData.js` – Mock dataset representing municipal documents

## How to run
npm install
npm run dev

## Purpose
The goal of this project is to demonstrate core UX flows using low-fidelity wireframes, improved wireframes, and a working prototype.
